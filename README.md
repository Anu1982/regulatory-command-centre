# Regulatory Copilot — Boehringer Ingelheim

Working repository for the Regulatory Copilot project.

## Purpose

Develop an AI copilot that helps Boehringer Ingelheim Regulatory Affairs teams draft, review, and manage submissions to global health authorities.

## Workshop

Customer workshop scheduled for tomorrow.

## Folder Layout

- [inputs/](inputs/) — Source materials (regulatory guidance, prior submissions, templates, meeting notes, decks).
- [docs/](docs/) — Curated documentation and generated artifacts, including the [Regulatory Intelligence AI Agent](docs/regulatory-intelligence-ai-agent.md) concept.
- [demo/](demo/) — Apple-style, no-build prototype of the Regulatory Intelligence AI Agent. Open [demo/index.html](demo/index.html) in a browser.
- [.github/copilot-instructions.md](.github/copilot-instructions.md) — Project context loaded automatically by GitHub Copilot.
- [AGENTS.md](AGENTS.md) — Quick start for coding agents.
- `.copilot-tracking/` — Session state for research, planning, and review workflows (created on first use).

## Getting Started

Upload files to the [inputs/](inputs/) folder and let me know when they are ready.

## Try the Demo

Open [demo/index.html](demo/index.html) in any modern browser (Safari, Chrome, Edge, Firefox). No build step, no dependencies. See [demo/README.md](demo/README.md) for what it demonstrates.
