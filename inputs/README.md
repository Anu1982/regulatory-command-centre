# Inputs

Place source files here for the Regulatory Copilot project. Suggested subfolders:

- `requirements/` — PRDs, BRDs, user stories
- `guidance/` — FDA, EMA, PMDA, ICH, and other health-authority guidance documents
- `templates/` — CTD modules, IND, NDA, MAA, and other submission templates
- `submissions/` — Redacted prior submissions and reference dossiers
- `meetings/` — Workshop transcripts and health-authority interaction notes
- `decks/` — Slides and presentations
